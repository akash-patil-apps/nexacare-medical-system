## How to check if a domain is available

1. **Instant Domain Search** (free, fast): https://instantdomainsearch.com  
   - Type the name (e.g. `pravaha`, `nexora`, `velara`).  
   - It shows .com, .in, .io, .health, etc. Green/“Available” = can register.

2. **Namecheap**: https://www.namecheap.com/domains/domain-name-search/  
   - Search the exact domain (e.g. pravaha.com).  
   - Shows price and availability.

3. **ICANN Lookup** (official): https://lookup.icann.org  
   - Use for final check: who owns it and when it expires.

4. **India:** For .in, use the same tools or registrars like BigRock, ResellerClub, or Namecheap (.in).

## If .com is taken: good alternatives

- **.in** — Strong for India; e.g. getpravaha.in, pravaha.in.
- **.health** — Fits healthcare; e.g. pravaha.health (check price; often premium).
- **get + name** — getpravaha.com, getnexora.com (common for apps).
- **try + name** — trypravaha.com, trynexora.com.
- **use + name** — usepravaha.com (less common).
- **name + app** — pravahaapp.com (clear but longer).

---

## Suggested workflow

1. Pick 3–5 names from the tables above (mix of Indian-rooted + coined if you like).
2. For each, check on **instantdomainsearch.com**:  
   `name.com`, `name.in`, and if you like `getname.com`.
3. If one has .com or .in available and you like the sound → lock that name and register the domain.
4. Optionally check trademark (e.g. India: https://ipindiaservices.gov.in for similar mark in class 44/42).
