# Custom Icons (PNG)

Place your PNG icon files in this directory:

1. **dashboard.png** - Dashboard icon from https://www.flaticon.com/free-icon/dashboard_1828673
2. **appointment.png** - Appointment icon from https://www.flaticon.com/free-icon/appointment_7322293
3. **prescription.png** - Prescription icon from https://www.flaticon.com/free-icon/prescription_482007
4. **lab.png** - Lab icon from https://www.flaticon.com/free-icon/tube_6992820

## How to Add Icons:

1. Download the PNG files from Flaticon
2. Save them in this directory (`/client/public/icons/`) with the exact names:
   - `dashboard.png`
   - `appointment.png`
   - `prescription.png`
   - `lab.png`

3. The icons will automatically be used in the sidebar menu

## Icon Requirements:

- Recommended size: 24x24px or 32x32px
- Format: PNG with transparent background
- The icons will be automatically colored (blue for selected, gray for unselected) using CSS filters
